# Screen Sound

Um projeto de console desenvolvido em C# para praticar lógica de programação, estruturas de repetição, listas e organização de métodos.

O sistema permite cadastrar e listar bandas em um menu interativo no terminal.

## Funcionalidades

- Menu interativo no console
- Cadastro de bandas
- Listagem de bandas cadastradas
- alidação de entrada do usuário
- Tratamento para lista vazia
- Avaliação de bandas
- Exibição de média por banda

## Tecnologias utilizadas

- C#
- .NET
- Console Application
- List
- Switch Case
- Métodos

## Conceitos praticados

- Estrutura básica de programas em C#
- Métodos e reutilização de código
- List<string>
- for
- if / else
- switch case
- int.TryParse()
- Organização de fluxo de menu
